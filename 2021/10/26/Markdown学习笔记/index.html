Markdown编辑快速入门（Typora)

Markdown标题

方法1：使用=和-标记一级标题以及二级标题

方法2：使用#号来表示几级的标题（几个#号就代表了几级标题）

Markdown 段落

Markdown的段落直接编辑填写就好，没有特殊的格式。段落的换行是使用两个以上的空格+回车。

2.1. Markdown的字体

使用*一左一右可以表示Markdown斜体 ：

斜体

使用**两左两右也可以表粗体：

粗体

使用***三左三右可以表示粗斜体：

粗斜体

（也可以将*替换为_，效果相同）

2.2. Markdown的分割线

你可以在一行中用三个以上的星号*、减号-、底线_来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格。(任意一种写法的效果都相同)

2.3. Markdown的删除线

只需要在文字两端分别加上两个波浪线~~即可：

BDMap

2.4. Markdown的下划线

使用HTML的u标签实现” “

效果如：下划线

2.5. Markdown的脚注

对文本补充说明：格式为: 【^ 要注明的文本】（得使用英文的中括号）

如此：Building Information Modeling ^1

Markdown列表

支持有序列表和无序列表

3.1. 无序列表

使用（*）、（+）、（-）作为列表标记，注意这些标记后加一个空格再填入内容，效果如下：

  • Internet of Things
  • Computer Vision

3.2. 有序列表

使用数字并加上一个点（.）来表示，也是需要打一个空格再键入：

  1. CityGML
  2. IFC

3.3. 列表嵌套

只需要在子列表的选项前添加四个空格即可：

  1. 第一件事 
    - 明天
- 后天
  2. 第二件事
    • 周三
    • 周六

Markdown区块

Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号，也可以见得区块可以套用，一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推：

区块引用

芜湖

如果在列表中使用区块：需要在>之前加上四个空格缩进。

  1. 第一个

    区块使用
    12345

Markdown代码

如果是段落上的一个函数或者代码片段可以用反括号包起来（`），这个就是esc下面那个按键：

printf("Hello world!");

如果是一个代码区块，使用4个空格或者1个制表符（Tab健）。也可以用```包裹一段代码，可以制定一种语言。比如我指定css：

Markdown图片

图片语法：

![alt 属性文本](图片地址)

![alt 属性文本](图片地址 “可选标题”)

因为markdown暂时还没有办法指定图片高度宽度，我们也可以通过一个img标签来实现插入图片：内容为

<img src=”https://img9.doubanio.com/view/photo/l/public/p2232705497.webp" width=”30%”/>

Markdown表格

1 2 3
4 5 6

这里比较推荐直接用Typora的工具生成表格。

Markdown的其他技巧

9.1. 多种HTML元素支持：

不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。

目前支持的 HTML 元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等 。

9.2. 转义

Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown 使用反斜杠\转义特殊字符：

**加粗字体**

\ 反斜线

* 星号

[] 方括号

9.3. 公式

Markdown 使用数学公式可以使用两个美元符号$$，分别在开头和结尾包裹TeX或者LaTeX格式的数学公式来实现。
